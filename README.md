# Secure-Voting-System

Welcome to the Secure-Voting-System! This program simulates a simple Secure-Voting-System that ensures security.

## Usage

1. Clone the repository:
    ```bash
    git clone https://github.com/NishchitSubedi/Secure-Voting-System
    ```

2. Run the program:
    ```bash
   python3 votingsystem.py
    ```
